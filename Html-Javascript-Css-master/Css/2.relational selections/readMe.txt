
Child:
A>B --> Selects any element B that is a direct child of A(ignores the deeper nested elements)

 
Adjacent Sibling:
A+B -->Selects the B element that is placed immediately after A element and shares the same parent.
        This selects only the immediate sibling.

 
General Sibling:
A ~ B -->Selects all the B elements that are placed after A element and share the same parent.
        This selects all the siblings.
